# Nano Banana 2

## 项目概述

Nano Banana 2 是一个 AI 驱动的图片生成包，可根据文字描述创建图片。支持文本生成图片和使用参考图片进行图片编辑。提供灵活的输出选项，包括多种格式、宽高比和分辨率。

## Block 功能

### Task Block: Nano Banana2 Image Generation

根据文本提示生成 AI 图片。支持使用参考 URL 进行图片编辑。输出图片 URL 和元数据，包括尺寸、文件大小和内容类型。

**主要输入：**
- `prompt` (必填)：期望图片的文字描述
- `image_urls` (可选)：用于编辑的参考图片 (1-14 个 URL)
- `num_images`：生成图片数量 (1-4)
- `aspect_ratio`：输出尺寸 (auto, 21:9, 16:9 等)
- `output_format`：JPEG、PNG 或 WebP
- `resolution`：0.5K 到 4K 质量

### Subflow Block: Nano Banana2 Download

生成图片并自动下载到本地文件。结合图片生成和批量下载功能。输出本地文件路径供后续处理。

## Block 组合建议

- **内容创作流程**：使用 Nano Banana2 Download 生成图片并保存到本地，然后连接图片处理 block 进行调整大小或格式转换。
- **批量生成**：使用不同的种子生成多张图片，然后使用数组 block 组织和处理结果。
- **基于参考图的编辑**：提供参考图片 URL 来创建现有图片的变体或风格转换。

## 基本用法

1. 输入描述性的文本提示（具体描述风格、主题和构图）
2. 可选择添加参考图片 URL 进行引导生成
3. 选择输出格式、宽高比和分辨率
4. 运行 block 生成图片
5. 使用返回的 URL 或文件路径进行展示或后续处理

## 示例

**文本生成图片：**
```
提示词："夕阳下宁静的山间湖泊，水面倒映着天空"
输出：图片 URL 数组
```

**使用参考图编辑：**
```
提示词："将这张照片转换为水彩画风格"
参考 URL：["https://example.com/photo.jpg"]
输出：风格化图片 URL
```

**下载生成的图片：**
```
使用 Nano Banana2 Download subflow
提示词："夜晚的未来城市天际线"
输出：已下载图片的本地文件路径
```
